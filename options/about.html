Manga Notifier

Created and designed by three college students within a span of 24 hours, Manga Notifier is a simple extension which allows a user to get notified whenever a new manga is released.

Special thanks to Pixel Perfect from Flat Icons for providing us the logo for the add-on and plugin and Materialize CSS team for providing us the Materialize CSS template.
